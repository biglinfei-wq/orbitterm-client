# Security Policy

## Supported Scope
- OrbitTerm client code in this repository.
- Installation artifacts and release metadata generated from this repository.

## Reporting
If you discover a security issue, do not publish exploit details directly.
Please contact the project maintainer through a private channel first.

Please include:
- Affected version
- Reproduction steps
- Impact assessment
- Suggested fix (optional)

## Notes
Commercial backend, admin controls, and entitlement policies are managed in
private server repositories and follow separate security procedures.
